The data explored in this practical is inspired by the data collected in the PRISM study(Program for Resistance, Immunology, Surveillance and Modelling of Malaria) in Uganda. The original dataset contained the optical density values for a single antigen (PfAMA1). The dataset used in this example follows similar population trends as those from PRISM, but contains multiple P. falciparum antigens from a multiplex dataset.

Demographic Data

The data explored in this practical is inspired by the data collected in the PRISM study (Program for Resistance, Immunology, Surveillance and Modelling of Malaria in Uganda).

Demographic variables within this dataset include:
- age: age: This is a continuous variable representing age in years.
- gender: This variable is split into Male and Female
- bednet: This variable shows whether the participant uses a bednet (1) or not (0).
- microscopy: This variable represents whether a participant had a positive microscopy test (1) for malaria or a negative malaria microscopy test (0).
- RDT: This variable represents a rapid diagnostic test (RDT) result. A positive test = 1, negative test = 0. - haemoglobin range: This variable represents the participant haemoglobin range in a categorical variable.

Age distributions and sero-conversion rates

The data in this study follows a common age distribution generally observed in rural sub-Saharan African populations.

Population age distributions will differ in South East Asia or Latin America, which may have an impact on the precision of the sero-conversion rate estimate because it is dependent on sero-prevalence by age category.

Step 2: Determining the seropositive cutoff - Finite Mixture Model

Finite Mixture Model and defining a seropositive cutoff

In this practical, we will define seropositivity using a Finite Mixture Model as the dataset lacks controls.

The normalmixEM function runs the finite mixture model for the data. This estimates a bi-model distribution (i.e.assumes two normal distributions in the data). The popular method of establishing the sero-positive cutoff is by adding 3 standard deviations (3 SD) to mean of the sero-negative curve.

Step 1: Fit the finite mixture model to each antigen MFI

If an antigen MFI response is larger than the sero-positive cutoff established by the FMM cutoff for that antigen, then the antigen response is considered sero-positive.

Historic exposure

AMA-1: FMM and seropositive cutoff

# Step 1: Fit 2-component Gaussian Mixture Model (FMM)
fmm_ama_1 <- normalmixEM(dat$AMA_1, k = 2) 
number of iterations= 65 
# Step 2: Use function to extract parameters
mixture_params_ama_1 <- mixture_params(fmm_ama_1)

# Step 3: Determine seropositive cutoff
cutoff_ama_1 <- mixture_params_ama_1 %>%
  filter(group == "Seronegative") # Establishing cutoff definition
cutoff_ama_1$three_sd
[1] 445.8657
## Creating variable for seronegative mean to be plotted in Step 5
mean_sero_neg_ama_1 <- mixture_params_ama_1 %>%
  filter(group == "Seronegative") # Establishing cutoff definition
sero_neg_mean_ama_1 <- mean_sero_neg_ama_1$mean

# Step 4: Adding a seropositive variable to the original dataset. Seropositive (1) or seronegative (0) based off the cutoff above 
dat[paste0("AMA_1", "_sero")] <- ifelse(dat$AMA_1 > cutoff_ama_1$three_sd, 1, 0)  # Define seropositivity

Step 3: Defining cumulative seropositivity

This section will create variables to determine whether a participant is considered seropositive based off of the antigen MFI value compared to the sero-cutoff established in the FMM models in the step above.

If an antigen MFI is above the threshold, then the participant is considered seropositive to that specific antigen.

Historic exposure

The code below creates the following variables:

seropos_antigen: - seronegative (0): the MFI value is below the MFI cutoff threshold established in the FMM above. - seropositive (1): the MFI value is equal to or greater than the MFI cutoff threshold established in the FMM above.

historic_cumulative: Combines the values of all historic antigens. The value is either - 0 = No antigen response is considered seropositive - 1 = One antigen MFI value is considered seropositive - 2 = Two antigen MFI value is considered seropositive - 3 = Three antigen MFI value is considered seropositive

seropos_historic: If the participant is seropositive to any of the 3 historic antigens (AMA-1 OR GLURP OR MSP1-19), they are characterised as seropositive to historic exposure.

As this study takes place in a high transmission region, the selection of antigens can be more lenient than in a low transmission region.

# Create cumulative seropositive variable
dat <- dat %>% 
  mutate(seropos_glurpr2 = ifelse(GLURP >= cutoff_glurp$three_sd, 1, 0), 
         seropos_pfama1 = ifelse(AMA_1 >= cutoff_ama_1$three_sd, 1, 0), 
         seropos_pfmsp119 = ifelse(MSP1_19 >= cutoff_msp1_19$three_sd, 1, 0))%>%
  mutate(
    historic_cumulative = seropos_pfmsp119 + seropos_pfama1 + seropos_glurpr2,
    seropos_historic = if_else(historic_cumulative >= 1, 1, 0)
  )

Estimating Force of Infection (FOI) using sero-conversion rates:

The Force of Infection (FOI) is a metric used to estimate transmission intensity of a disease, in this case malaria. It represents the rate a susceptible population acquires infection.

##Historic exposure ### Step 1: Estimate sero-conversion rate (force of infection) - lambda - sero-conversion rate with 95% confidence intervals - rho - sero-reversion rate with 95% confidence intervals

# Individuals ages less than  1 year excluded due to maternal antibodies #
df <- dat %>% 
  filter(age > 0)

# Step 1: Create a structured data object that organizes age, seropositivity status, and grouping information in a standard format 
data2 <- create.data.object(round(df$age), df$seropos_historic)

# Step 2: Estimating sero-conversion rate by fitting a reversible catalytic model
scr_fit <- simple.rcm.analysis(data2, analysis='overall', int.rho=c(0.001,0.250))

Fitting seroconversion rate with 2 forces of infection

This step allows the model to account for different transmission levels: historic and current. This change in transmission intensity over time could be because of a drop in malaria burden due to control interventions

  • set time.common to the age where you can see a potential change point in your data
  • lambda1 represents the historical transmission intensity (estimated for individuals born before time of change)
  • lambda2 represents the current transmission intensity (estimated for individuals born since time of change)
# Fit sero-conversion rate to under 15 years and 15 and older
scr_fit2 <- two.rcm.analysis(data2, analysis="split-shared-rho",int.rho=c(0.001,0.005),time.common=15)
par(mfrow=c(1,1))

# Plot
plot_scr2(scr_fit2$estimates,plot.age.profile=seropos_historic.age.profile,district="High-transmission 2-step force of infection: Historic exposure",colour="#E4A804")
